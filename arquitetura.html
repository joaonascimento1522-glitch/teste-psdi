Arquitetura de Alto Nível

Componentes propostos para implementar o PSDI com segurança e escalabilidade

Mobile

App offline-first (React Native / Flutter), SQLite criptografado, fila de sincronização, mapas OSM, câmera e biometria nativas.

Backend & APIs

API Gateway (REST/GraphQL), microsserviços (prontuário, agendamento, notificações), camada FHIR para interoperabilidade e webhooks para integrações.

Infra & Segurança

PostgreSQL, Redis, S3, Keycloak (OAuth2/OIDC), TLS 1.2+, TDE, RBAC, MFA, trilhas de auditoria e backups automáticos.

Exemplo: Modelo mínimo de dados (Postgres)

patients (id, cns, cpf, nome, nascimento, sexo, contatos, endereco_json, consentimento_json, created_at)
households (id, geo_lat, geo_lng, endereco, saneamento_json, renda, created_at)
encounters (id, patient_id, tipo, local, profissional_id, data_hora, resumo)
